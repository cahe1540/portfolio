React Search UI

This project is a search UI for the Punk Beer API. It allows a user to search data by name from the API and displays the results as cards. The user can store favorites and can navigate away and return to the website and recover their favorites. This app is made using React, HTML, and CSS. View the source code on github.

This is a leaderboard app for the video game, Age of Empires II DE. This project was made using Javascript, HTML and CSS. The app works by submitting a fetch request to an open source Age of Empires II API. Data is consumed and rendered to a UI. View the source code on github.
